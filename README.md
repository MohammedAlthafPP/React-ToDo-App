# Todo App - React

A Todo Application in React.js

## Features

- Add , edit, delete todo
- Responsive


## Tech Stack

- React.js


## Run 

Clone the project

```bash
  git clone https://github.com/MohammedAlthafPP/React-ToDo-App.git
```

Go to the project directory

```bash
  cd React-ToDo-App
```

Install dependencies

```bash
  npm install
```

Start 

```bash
  npm run start
```
